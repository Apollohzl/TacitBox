# 默契盒子 (TacitBox)

## 项目简介
默契盒子（TacitBox）是一个参考微信付费游戏小程序开发的Web应用。主要通过A用户进行出题，其他用户回答问题，然后计算出相互的默契度，并进行排名，从而促进朋友之间的互动和交流，增加了趣味性和互动性，也可以帮助用户更好地了解自己的兴趣爱好和特点，有助于增进彼此之间的了解和认识。

## 技术栈
- Next.js 14
- React
- TypeScript
- Node.js
- MongoDB Atlas (数据库)

## 项目架构
- 服务端渲染 (SSR) + 静态生成 (SSG)

## 运行项目
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 构建项目：`npm run build`
4. 启动生产服务器：`npm start`
